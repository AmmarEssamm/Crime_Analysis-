# Crime_Analysis
### Overview
Analyzed crime incident data to identify trends, high-risk areas, and temporal patterns
using Python.

### Tools
- Python
- pandas, NumPy
- matplotlib, seaborn

### Key Skills Demonstrated
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Insight Generation

### Outcome
Delivered clear insights that demonstrate the ability to transform raw data into
actionable information.
